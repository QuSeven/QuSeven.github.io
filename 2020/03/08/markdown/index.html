一些jupyter notebook的快捷键操作和Markdown的一些语法介绍

jupyter notebook

  • Esc退出当前指定的单元格内
  • up and down上下移动想要选中的单元格就按enter进入
  • A 或 B 向上或向下增添一个代码单元格
  • M调出Markdown单元格
  • Y调出代码单元格
  • Shift+up or down 连续选中单元格
  • D+D 删除
  • Z 将撤销单元格删除

Markdown

Mrakdown是一种轻量化的、易于学习的标记语言，用于格式化纯文本。它的语法与HTLM标记有一对一的对应关系。
